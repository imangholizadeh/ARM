# LPC1768-SPI_FLASH_W25QXX
I hope use it and enjoy. 
I use LPC1768 and Keil Compiler. 

ARM SPI Configuration
------------------------
1) P0.7  -->   SCK
2) p0.6  -->   CS
3) p0.8  -->   MISO
4) p0.9  -->   MOSI

ARM Serial Configuration
------------------------
3) p0.2  -->   TXD
4) p0.3  -->   RXD

W25QXX SPI Configuration
------------------------
1) P6  -->   SCK
2) P1  -->   CS
3) p2  -->   MISO
4) p5  -->   MOSI
